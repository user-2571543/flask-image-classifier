# 🖼️ Flask Image Classifier - 発表資料

## 1. システムの概要

このシステムは、**ResNet18** という AI モデルを使用して、ユーザーがアップロードした画像を以下の3つに分類するWeb アプリケーションです。

- 🐕 **犬（Dog）**
- 🐈 **猫（Cat）**
- ❓ **その他（Other）**

---

## 2. 主な機能

### 🔒 ユーザー認証
- ユーザー登録・ログイン機能
- パスワードは bcrypt でセキュアに保護

### 📸 画像アップロード（ログイン必須）
- JPG, PNG, GIF 形式に対応（最大16MB）
- ドラッグ&ドロップまたはクリックで簡単アップロード
- アップロード時に自動的に AI が分類

### 🧠 AI 分類（ログイン必須）
- ResNet18 を使用した画像分類
- 分類ラベル：犬 / 猫 / その他
- 信頼度を0～100%で表示

### 📊 ギャラリー表示（ログイン必須）
- アップロードした全画像を一覧表示
- 分類結果と信頼度を表示
- ページネーション対応

### 🗑️ 削除機能（ログイン必須）
- 不要な画像と分類結果をまとめて削除
- データベースから自動削除

## 4. AI モデルについて

### ResNet18 とは

- **学習データ**: ImageNet（100万枚以上の画像）
- **学習済みモデル**: PyTorch torchvision から提供
- **出力**: 1000クラスの確率値
- **用途**: 汎用的な物体認識

### 入力・出力

**入力:**
```
ユーザーがアップロードした画像ファイル
（JPG, PNG, GIF形式、最大16MB）
```

**処理:**
```
1. 画像を 224×224 ピクセルにリサイズ
2. ResNet18 で推論
3. 確率が最も高いクラスを取得
4. そのクラスが犬/猫/その他かを判定
```

**出力:**
```
分類ラベル: "dog" または "cat" または "other"
信頼度: 0.0～1.0（例：0.95 = 95%）
```

### 分類の仕組み

```
ResNet18の出力 
    ↓
1000クラスの確率値を取得
    ↓
最高確率のクラスを選択
    ↓
そのクラスIDを確認
    ↓
犬クラス（151-169）？ → "dog"
猫クラス（281-295）？ → "cat"
それ以外？ → "other"
    ↓
分類結果を表示
```

## 5. データベース構成

### 3つのテーブル

**User テーブル**
```
- id: ユーザーID
- username: ユーザー名（一意）
- password_hash: パスワードハッシュ
- created_at: 登録日時
```

**UserImage テーブル**
```
- id: 画像ID
- user_id: ユーザーID（外部キー）
- filename: 元のファイル名
- upload_path: 保存先パス
- uploaded_at: アップロード日時
```

**Prediction テーブル**
```
- id: 予測ID
- user_image_id: 画像ID（外部キー）
- label: 分類ラベル（dog/cat/other）
- confidence: 信頼度（0.0～1.0）
- created_at: 予測日時
```

**関係性:**
```
User
  ↓ 1:多
UserImage
  ↓ 1:1
Prediction
```

## 7. セットアップ・実行方法

### インストール
```bash
# リポジトリクローン
git clone https://github.com/user-2571543/flask-image-classifier.git
cd flask-image-classifier

# 仮想環境作成
python -m venv venv
source venv/bin/activate  # macOS/Linux
# または venv\Scripts\activate  # Windows

# 依存パッケージインストール
pip install -r requirements.txt

# データベース初期化
flask init-db
```

### 実行
```bash
python app.py
# http://localhost:5000 にアクセス
```
